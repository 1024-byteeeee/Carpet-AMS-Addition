# New Rules

- `fakePlayerPickUpController` Two modes are provided to control whether fake player can pick up items.

  - [MainHandOnly] - Only the main hand can pick up items

  

  - [NoPickUp] - Unable to pick up items at all

  

  - [false] - Disable rule
