# 新增规则

- 无限耐久（infiniteDurability）

  

  - 使用物品不会丢失耐久。



- 欢迎消息（welcomeMessage）

  

  - 在玩家进入服务器时发送一条自定义消息给玩家。



- 获取存档大小（commandGetSaveSize）

  

  - 使用 “/getSaveSize” 命令获取当前存档大小。
