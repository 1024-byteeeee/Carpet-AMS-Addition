# 新规则

- **潜影贝傀儡（shulkerGolem）**

  可以通过在潜影盒上放置一个雕刻南瓜来召唤一只潜影贝。
  
- **猎头（headHunter）**

  当玩家死亡时他们会掉落一个自己的头颅。
  
- **获取玩家头颅（commandGetPlayerSkull）**
  
  使用 “/getPlayerSkull” 命令来获取一名在线玩家的头颅



# 更改

- 移除了 `开启1.21实验性内容合成配方（enableMinecraft1_21ExperimentalRecipes）` 规则。
